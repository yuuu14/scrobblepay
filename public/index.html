🎯 How It Works

1. Connect your Last.fm account
2. An AI agent reads your scrobble history
3. It calculates: $5 ÷ your plays per artist
4. Sends nanopayments via Circle CLI on Arc L1 ⚡
5. Artists receive USDC directly — no middleman

Press "Load Split" to see your data 🎵

⚡ Try It Yourself

# Install
uv tool install git+https://github.com/the-canteen-dev/ARC-cli
npm install -g @circle-fin/cli

# Run the agent
npm run agent -- --user YOUR_LASTFM_USER --budget 5.0

# With real payments
circle wallet create
npm run agent -- --user YOUR_LASTFM_USER --budget 5.0 --execute
